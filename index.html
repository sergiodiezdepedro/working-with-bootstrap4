Bootstrap

Las cards no tienen margin por defecto. Asimismo, tampoco tienen un ancho definido por defecto, por lo que ocupan completamente el ancho disponible de su elemento padre. Se puede personalizar fácilmente con el siguiente atributo: style="width: 18rem". Obviamente, el 18 es un ejemplo de una cifra.

Imágenes Responsive

Las imágenes se convierten en responsive cuando se añade la clase .img-fluid.

Figure

Las figures proporcionan cierto estilo a las etiquetas propias de este elemento HTML. Las imágenes requieren la clase pertinente para convertirlas en responsive. No tienen un tamaño definido de forma explícita.

Breadcrumb

  1. Home
  2. Sección 1
  3. Subsección X

Puesto que las migas de pan proporcionan una navegación, es una buena idea añadir una etiqueta descriptiva como aria-label="breadcrumb" para describir el tipo de navegación que proporciona el elemento <nav>, así como aplicar aria-current="page" al último ítem del conjunto para indicar que señala la página actual.
